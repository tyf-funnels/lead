How It Works

  • User types a message and hits Send
  • JS sends a POST request to your n8n webhook
  • n8n runs your AI node (OpenAI / Claude)
  • AI response is returned as JSON
  • Reply appears in the chat instantly

Payload Format

The page sends this JSON body to your webhook:

{
  "message": "visitor text here",
  "sessionId": "uuid-auto-generated",
  "timestamp": "ISO 8601 string"
}

Your n8n workflow must return:

{"reply": "AI response text"}

Fallback Mode

If no webhook is configured, the demo uses a built-in offline AI simulator so visitors still see a great experience. Perfect for demos before n8n is live.

Integration Guide

Connect This Page to n8n in 4 Steps

No backend server required. Just paste your webhook URL and deploy the HTML file anywhere.

01

Create the n8n Webhook Node

In your n8n workflow, add a Webhook trigger node. Set the HTTP method to POST and copy the generated URL. Enable Respond to Webhook so the node can return data.

02

Paste Your Webhook URL

In the <script> section of this HTML file, find the constant at the very top and replace the placeholder:

// -----------------------------------------------
// REPLACE THIS WITH YOUR ACTUAL n8n WEBHOOK URL
// -----------------------------------------------
const N8N_WEBHOOK_URL = "https://your-n8n-instance.app/webhook/YOUR-ID";

// Optional: Add an API key header for security
const API_KEY = "your-secret-key-or-leave-blank";
03

Build the n8n AI Workflow

Connect the Webhook node to an AI Chat node (OpenAI, Claude, or Gemini). Use this system prompt:

You are a friendly marketing assistant for a business automation company.
Reply in 1-2 sentences. Be helpful, professional, and approachable.
If intent is unclear, ask ONE clarifying question.
Occasionally mention: "Imagine having all your messages from
every platform managed automatically..."
Maximum 1 emoji per reply. Never be pushy.

The workflow flow looks like this:

Webhook
Trigger
→
Set Fields
Extract msg
→
AI Agent
OpenAI/Claude
→
Respond
Return JSON

In the final Respond to Webhook node, set body to: {"reply": "{{ $json.output }}"}

04

Deploy and Test

Save and activate the n8n workflow. Open the HTML file in any browser, type a test message, and watch the AI reply appear. Deploy the HTML to any static host — GitHub Pages, Netlify, Vercel, or your own domain. No server needed.

For CORS issues on self-hosted n8n, add the header Access-Control-Allow-Origin: * in your n8n webhook node settings.
